
# Project 1 - E-commerce Sales Performance Dashboard

## Tools Used
Power BI Desktop

## Dataset
Sample Superstore Dataset (Kaggle)
Link: https://www.kaggle.com/datasets/vivek468/superstore-dataset-final

## Dashboard Structure
### Page 1 - Sales Dashboard
- KPI Cards: Total Revenue, Total Profit, Total Orders, Profit Margin %
- Line Chart: Sales trend by Month/Year
- Bar Chart: Top 10 Products by Sales
- Pie Chart: Sales by Region
- Donut Chart: Sales by Segment
- Slicers: Year, Region, Category

### Page 2 - State Analysis
- Treemap: Sales by State (full page)

### Page 3 - Detailed Insights
- Table: Product Name, Sales, Profit, Quantity
- Card: Average Order Value

## Key Insights
- Technology category drives highest revenue
- West region has strongest sales performance
- Consumer segment contributes highest sales share
- Top products are concentrated in Technology category
- Q4 shows consistent sales peak every year

## Viva Answers
**Define ROI in marketing?**
ROI (Return on Investment) measures how much revenue
is generated for every dollar spent on marketing.
Formula: ROI = (Revenue - Cost) / Cost × 100
Example: Spent $100 on ads, earned $300 revenue → ROI = 200%

**What is a pivot table?**
A pivot table is a data summarization tool that automatically
sorts, counts, and totals data. It helps analyze large datasets
by grouping and aggregating values.
Example: Sales grouped by Region and Category to see
which combination performs best.
